# 🍽 Spice Garden Restaurant Website

## Project Overview

Spice Garden Restaurant Website is a frontend web project created to provide an
online restaurant experience. This website displays restaurant details, menu
items, services, and allows users to place food orders.

This project is developed as a student web development project using HTML, CSS,
and JavaScript.

## Technologies Used

- HTML
- CSS
- JavaScript

## Features

- Attractive restaurant homepage
- About restaurant section
- Food menu with categories
- Starters section
- Rice items section
- Main course section
- Desserts section
- Services section
- Order Now option
- Table booking
- Customer feedback
- Contact section
